# 编辑距离

## 问题描述

给定两个字符串 `word1` 和 `word2`，计算出将 `word1` 转换成 `word2` 所使用的最少操作数。

你可以对一个字符串进行如下三种操作：
1. **插入**一个字符
2. **删除**一个字符
3. **替换**一个字符

**例如：**
```
示例 1:
输入：word1 = "horse", word2 = "ros"
输出：3
解释：
horse -> rorse (将 'h' 替换为 'r')
rorse -> rose (删除 'r')
rose -> ros (删除 'e')

示例 2:
输入：word1 = "intention", word2 = "execution"
输出：5
解释：
intention -> inention (删除 't')
inention -> enention (将 'i' 替换为 'e')
enention -> exention (将 'n' 替换为 'x')
exention -> exection (将 'n' 替换为 'c')
exection -> execution (插入 'u')

示例 3:
输入：word1 = "", word2 = "a"
输出：1
解释：插入一个字符 'a'
```

**关键问题：**
- 字符串是否可能为空？（可能）
- 操作的代价是否都为 1？（是）
- 是否需要输出具体的操作步骤？（不需要，只返回最少操作数）
- 是否区分大小写？（区分）
