# DreamFace App 项目面试题目 (30题)

## 技术架构类 (8题)

### 1. 多页面架构设计
**Q: 你提到实现了41个页面独立打包发布的多页面架构，能详细说说这个架构的设计思路和实现方案吗？**
- 考察点：项目架构设计能力、模块化思维

### 2. 代码分割策略
**Q: 页面级代码分割是如何实现的？相比单页应用有什么优势和劣势？**
- 考察点：性能优化、打包策略、SPA vs MPA理解

### 3. Vue 2.7 + TypeScript
**Q: 为什么选择Vue 2.7而不是Vue 3？在Vue 2.7中使用TypeScript遇到了哪些挑战？**
- 考察点：技术选型考虑、版本差异理解

### 4. 状态管理方案
**Q: 在多页面架构下，Pinia的状态是如何管理的？页面间数据如何共享？**
- 考察点：状态管理理解、跨页面通信

### 5. 移动端适配
**Q: 面向海外用户的移动端App在适配方面有什么特殊考虑？**
- 考察点：国际化意识、移动端开发经验

### 6. 性能优化策略
**Q: 你提到首屏时间下降40%，具体采用了哪些优化手段？**
- 考察点：性能优化实战经验

### 7. 工程化规范
**Q: ESLint/Prettier/Husky的配置和团队规范是如何制定的？**
- 考察点：团队协作、代码规范制定

### 8. 组件库设计
**Q: 组件库复用是如何实现的？如何保证组件的通用性和扩展性？**
- 考察点：组件设计思维、抽象能力
