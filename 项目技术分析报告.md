## 🏗️ 技术架构特色

### 1. 独创的多页面架构设计
```
架构模式: Multi-Page Application (MPA) + 领域驱动设计
页面隔离: 每个页面独立打包、独立部署
代码分割: 自动按页面进行代码分割，优化加载性能
```

**技术亮点**:
- 41个独立页面模块，避免单页面应用的复杂度
- 基于`vue.mulitpage.js`的自动页面发现机制
- 每个页面都有独立的入口点和HTML模板

### 2. 混合应用架构
```
技术栈: Vue 2.7 + TypeScript + Vant + Pinia + Tailwind CSS
运行环境: WebView + Native Bridge通信
设备适配: 移动端优先，375px基准设计
```

### 3. 核心技术栈选择理由

| 技术选择 | 理由 | 面试加分点 |
|---------|-----|-----------|
| Vue 2.7 | 平衡了Vue2稳定性与Vue3新特性 | 技术选型的平衡思维 |
| TypeScript | 大型项目类型安全，提升开发效率 | 工程化意识 |
| Vant | 成熟的移动端UI库，减少开发成本 | 合理利用生态资源 |
| Pinia | 新一代状态管理，比Vuex更简洁 | 技术前瞻性 |
| Tailwind | 原子化CSS，快速UI开发 | 现代化CSS工程化 |
